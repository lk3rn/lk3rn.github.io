这篇文章主要是记录自己在搭建博客时遇到的坑，如果你也想搭建一个博客你可以参考这篇文章，写的很细我也就不多说了。

0x00 密钥配置

密钥在本地配置时，需要通过ssh-keygen -t rsa -C “你注册github时的邮箱地址”命令进行生成，当时在本地并没有找到生成的密钥文件，因为我是在windows环境下创建的，生成的密钥文件有隐藏属性，但是自己进行设置后，还是没有找到文件。后来发现在生成的密钥文件后面有提示路径，我们只要以.pub后缀的路径复制粘贴到文件系统的路径输入框中，再将路径改成在windows下的路径格式即可，然后参考上面的文章将公钥传到GitHub上面即可。例如下图中的图片改成c:/Users/lenovo/.ssh/id_rsa.pub。

0x01 配置文件的正确配置

自己到最后关键步骤时运行hexo d命令没有反应，也没有提示，刚开始也没有想到去找度娘，就自己在哪按照上面的步骤从来一下，但是并没有什么作用，仍旧没有反应，后来直接在百度上搜索“hexo d命令”没有反应，结果发现是自己的配置文件没有配对。（感觉真的很坑，以后一定要仔细）原来在下图自己添加信息的地方冒号后面需要将两个空格，且一定要冒号是英文的，如果你要用的是Visual Studio打开那么颜色显示应该和下面的一样。

0x02 总结

以后写配置文件时一定要记着将文件里的符号写成英文的，且注意格式问题。还有一定要记着看操作时给的错误信息。

这篇文章写的不是很详细，因为是自己初次发表文章以后自己一定会努力，发布一些更好的文章，另外感谢学长耐心的指导。
